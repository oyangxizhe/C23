# C23

#### 介绍
本软件为给苏州某汽车零部件公司开发的进销存管理系统
软件除了基本的进销存功能外，主要是加了委外加工的部分即通过委外加工领料单与委外加工入库单来算委外加工费
及通过录入计件工资入库单来算员工计件工资。
